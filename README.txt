# Jeu de Démineur

Bienvenue dans le jeu de Démineur !

## Description

Le Démineur est un jeu de réflexion dans lequel le joueur doit découvrir toutes les cases d'une grille qui ne contiennent pas de mines, sans faire exploser ces dernières.

## Fonctionnalités

- Différents niveaux de difficulté (facile, moyen, difficile)
- Interface utilisateur intuitive
- Sauvegarde et chargement de parties
- Statistiques de jeu

## Installation

1. Clonez ce dépôt sur votre machine locale :
    ```
    git clone https://github.com/votre-utilisateur/votre-repo.git
    ```
2. Accédez au répertoire du projet :
    ```
    cd votre-repo
    ```
3. Installez les dépendances requises :
    ```
    pip install -r requirements.txt
    ```

## Utilisation

1. Lancez le jeu :
    ```
    python demineur.py
    ```
2. Suivez les instructions à l'écran pour jouer.
